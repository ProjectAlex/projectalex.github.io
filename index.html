Literature Survey

The Game

  • The game is an adaptation of "Snakes and Ladders" and "Scene it"
  • The game can be played by as many players as we want.
  • A question appears on the screen, and all the participants are supposed to answer the question
  • If a person seems to be doing well in one particular field, the questions would be from some other field. (Machine Learning)
  • The question will be a combination of pictures, videos, audio clips in various topics to exploit the fact that the game is digital.
  • The first person to reach the finish point is declared winner.

For eg.

  • For a 5 player game if 3 people answer the question quicker than the player playing his hand he moves 2 spaces ahead, or if he is the first to answer he moves 5 spaces ahead.
  • This keeps all the players engrossed as they are constantly interacting with the game.

Existing Solutions and Problems

  • Google's Chromecast is not opensource and thus does not give enough flexibility.
  • PiCast runs on a raspberry Pi and allows devices to content stream over WiFi. It is priced similar to the chromecast making it economically feasible. Devices communicate with PiCast using an API. It supports only Videos, Audio, Images like Chromecast
  • Another device we considered was Roku Stick which uses DLNA to stream content and can be plugged into any HDMI compatible TV.

Challenges we face

  • Speed - The connection speed may cause problems during gameplay.
  • Maximum number of players the server can handle will be a limiting factor.
  • Should be interconnected using WiFi
  • Fluctuations in speed and connections

How do we plan to tackle
the challenges

  • The game must be tweaked according to the connection speed that is achievable. For eg a turn based game for a slow connection or a fast everyone-plays multiplayer game if the connection is fast enough
  • The upper limit for number of players should be set based on the servers capability and the game that is chosen
  • The app must be able to work under a wobbly connection allowing players to reconnect and continue and not get kicked out
